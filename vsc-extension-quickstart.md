# Legacy: VS Code extension source

This repo is **oklch-skill**, a Cursor skill. The primary content is in `skills/oklch/`.

The **VS Code extension** (OKLCHanger) source is still in `src/` for reference. To build or run the extension:

- Install dependencies: `npm install`
- Compile: `npm run compile`
- Run from VS Code: use the Run and Debug view and launch "Extension"

The extension is not the main product of this repository. For the Cursor skill, see [README.md](../README.md) and `skills/oklch/SKILL.md`.
